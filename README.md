# StandbyDashboard

iPad向けのスタンバイ時計 + 音楽ダッシュボードアプリ

## 機能

- 時計表示
- 音楽の再生 / 一時停止
- シークバー操作
- 前の曲 / 次の曲
- ニュースモードとの切り替え
- 焼き付き防止デザイン

## 環境

- iPad
- iPadOS 17以上 (多分)
- Swift Playgrounds

## こだわり

### 焼き付き防止

- 時計デザインを10分ごとに変更
- 2曲ごとにコントロールUIを変更

### その他

- ニュースモード切り替え
- アルバムアート表示 / 非表示切り替え

## 動かし方

1. iPadに Swift Playgrounds をインストール
2. 新規アプリを作成
3. プロジェクトを開く
4. コードを貼り付ける
5. 実行！
