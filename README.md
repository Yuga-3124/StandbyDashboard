# iPad用音楽プレーヤー

## 説明
iPad用のSwiftで動く系の音楽プレーヤーです。

## 機能

- 時計
- 音楽の再生/一時停止
- 音楽のシークバーの移動
- 音楽の前の曲/次の曲
- ニュースモードとの切り替え
- 焼きつき防止のためのグラフィックデザイン

## 環境
iPad
iPadOS17以上 多分

## こだわり

### 焼きつき防止
- 時計を10分ごとにデザインを変更
- 2曲ごとにコントロールセンターのデザインを変更

### 他
- ニュースモードとの切り替え
- アルバムアートの表示/非表示の切り替え

## 注意
AIを使用して開発しています。コード品質は保証できません。
