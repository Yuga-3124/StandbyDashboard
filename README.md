# StandbyDashboard

iPad向けのスタンバイ時計 + 音楽ダッシュボードアプリ

## 機能

- 時計表示
- 音楽の再生 / 一時停止
- シークバー操作
- 前の曲 / 次の曲
- ニュースモードとの切り替え
- 焼き付き防止のため、定期的にUIを変化

## 環境

- iPad
- iPadOS 17以上
- Swift Playgrounds

## こだわり

### 焼き付き防止

- 時計デザイン：10分ごとに変化
- コントロールUI：2曲ごとに変化

### その他

- モード切り替え（音楽 / ニュース）
- 表示切り替え（アルバムアート）

## 動かし方

1. iPadに Swift Playgrounds をインストール
2. 新規アプリを作成
3. プロジェクトを開く
4. コードを貼り付ける
5. 実行！
